# NewBets - Self-Learning AI Betting System

A Progressive Web App (PWA) that uses AI to analyze sports betting opportunities across multiple sports.

## Setup

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run build:css` to build Tailwind CSS
4. Open `ai-betting-system.html` in a browser or deploy to GitHub Pages
